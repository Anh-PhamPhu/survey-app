## Requirements
You need to have PHP version **8.0** or above. Node.js version **12.0** or above.
## Installation
### Back-end
```
$ cd survey-app
$ cd laravel-vue-survey
$ composer install
$ composer update
```
### Front-end

```
$ cd survey-app
$ cd vue
$ npm install
$ npm run dev
```
